Complex Numbers

Definition

A complex number is a number that can be expressed in the form a + bi, where a and b are real numbers, and i is the imaginary unit satisfying i² = -1.

Standard Form: z = a + bi
Where: a = real part, b = imaginary part, i = √(-1)

The Imaginary Unit

The imaginary unit i is defined as:

i² = -1, so i = √(-1)

Representation Forms

Rectangular Form

z = a + bi

Polar Form

z = r(cos θ + i sin θ) = r e^(iθ)

Operations

Addition

(a + bi) + (c + di) = (a + c) + (b + d)i

Subtraction

(a + bi) - (c + di) = (a - c) + (b - d)i

Multiplication

(a + bi)(c + di) = (ac - bd) + (ad + bc)i

Division

(a + bi) / (c + di) = [(a + bi)(c - di)] / (c² + d²)

Complex Conjugate

z̄ = a - bi

Magnitude

|z| = √(a² + b²)

Example: Adding

(3 + 4i) + (2 - 5i) = 5 - i
