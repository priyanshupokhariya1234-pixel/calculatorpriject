Calculus

Introduction

Calculus is the branch of mathematics that studies continuous change. It has two main branches: Differential Calculus (derivatives) and Integral Calculus (integrals).

Differential Calculus (Derivatives)

Derivatives measure the instantaneous rate of change of a function.

Definition

f'(x) = lim(h→0) [f(x+h) - f(x)] / h

Common Derivatives

  • d/dx(xⁿ) = nxⁿ⁻¹
  • d/dx(sin x) = cos x
  • d/dx(cos x) = -sin x
  • d/dx(eˣ) = eˣ
  • d/dx(ln x) = 1/x

Applications

  • Velocity and acceleration
  • Optimization
  • Curve behavior

Integral Calculus (Integration)

Integration is the reverse of differentiation.

Definition

∫ f(x) dx = F(x) + C

Common Integrals

  • ∫ xⁿ dx = xⁿ⁺¹/(n+1) + C
  • ∫ sin x dx = -cos x + C
  • ∫ cos x dx = sin x + C
  • ∫ eˣ dx = eˣ + C
  • ∫ 1/x dx = ln|x| + C

Applications

  • Area under curves
  • Volumes
  • Distance traveled

The Fundamental Theorem of Calculus

∫[a to b] f(x) dx = F(b) - F(a)

Limits

lim(x→a) f(x) = L

Chain Rule

d/dx[f(g(x))] = f'(g(x))·g'(x)

Product Rule

d/dx[f(x)g(x)] = f'(x)g(x) + f(x)g'(x)

Quotient Rule

d/dx[f(x)/g(x)] = [f'(x)g(x) - f(x)g'(x)] / g(x)²
