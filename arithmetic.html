Arithmetic

Introduction

Arithmetic is the branch of mathematics dealing with basic operations: addition, subtraction, multiplication, and division. It forms the foundation for all higher mathematics.

Basic Operations

Addition (+)

Combining two or more numbers to get a sum.

a + b = c

Properties:

  • Commutative: a + b = b + a
  • Associative: (a + b) + c = a + (b + c)
  • Identity: a + 0 = a

Subtraction (-)

Finding the difference between two numbers.

a - b = c

Note: Subtraction is not commutative (a - b ≠ b - a)

Multiplication (×)

Repeated addition or scaling.

a × b = c

Properties:

  • Commutative: a × b = b × a
  • Associative: (a × b) × c = a × (b × c)
  • Distributive: a × (b + c) = a × b + a × c
  • Identity: a × 1 = a
  • Zero: a × 0 = 0

Division (÷)

Splitting into equal parts or finding how many times one number fits into another.

a ÷ b = c (where b ≠ 0)

Important: Division by zero is undefined!

Order of Operations (PEMDAS)

To ensure consistent results, follow this order:

  1. Parentheses (and brackets, braces)
  2. Exponents (powers and roots)
  3. Multiplication and Division (left to right)
  4. Addition and Subtraction (left to right)

Example

Calculate: 3 + 4 × 2² - (6 ÷ 2)

  • Parentheses: 6 ÷ 2 = 3
  • Exponents: 2² = 4
  • Multiplication: 4 × 4 = 16
  • Addition/Subtraction: 3 + 16 - 3 = 16

Working with Decimals

Addition and Subtraction

Line up decimal points vertically, then add or subtract as usual.

Example: 12.34 + 5.678

  • Line up: 12.340 + 5.678 = 18.018

Multiplication

Multiply as whole numbers, then count total decimal places in both numbers.

Example: 2.5 × 0.3

  • 25 × 3 = 75
  • Total decimal places: 1 + 1 = 2
  • Answer: 0.75

Division

Move decimal point in divisor to make it whole, move same places in dividend, then divide.

Working with Fractions

Adding/Subtracting Fractions

Find common denominator, then add/subtract numerators.

a/b + c/d = (ad + bc) / bd

Multiplying Fractions

(a/b) × (c/d) = (a × c) / (b × d)

Multiply numerators and denominators separately.

Dividing Fractions

(a/b) ÷ (c/d) = (a/b) × (d/c) = (a × d) / (b × c)

Multiply by the reciprocal (flip the second fraction).

Percentages

Percent means "per hundred".

Percentage = (Part / Whole) × 100%
Part = (Percentage / 100) × Whole

Example: What is 25% of 80?

  • Part = (25/100) × 80 = 0.25 × 80 = 20

Tips for Accuracy

  • Always follow PEMDAS order of operations
  • Check your work by doing operations in reverse
  • Group decimals together when adding/subtracting
  • Estimate first to catch obvious errors
  • Double-check decimal point placement
  • Break long calculations into smaller steps

Common Mistakes to Avoid

  • Forgetting order of operations
  • Misplacing decimal points
  • Dividing by zero
  • Not finding common denominators for fractions
  • Forgetting to carry or borrow in addition/subtraction

Mental Math Tricks

  • Multiplying by 10: Add a zero (or move decimal right)
  • Multiplying by 5: Multiply by 10, then divide by 2
  • Multiplying by 11: For two-digit numbers, add digits and put sum in middle
  • Percentages: 10% = divide by 10, 50% = divide by 2
